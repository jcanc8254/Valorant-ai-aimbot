## 🔍 Troubleshooting Common Issues

If the program fails to run, consult this list of fixes:

* Software Does Not Open: Ensure your graphics drivers are updated to the latest version. Visit the website of your card manufacturer to download the installer.
* Aiming Feels Jumpy: Close other programs running in the background. High CPU usage causes lag in the detection logic.
* Target Not Detected: Check your game settings. The program works best in Fullscreen or Borderless Windowed mode. Ensure your game resolution matches your desktop resolution.
* Anti-Virus Alerts: Some security software flags these programs as false positives because they hook into mouse input. Add an exclusion in your security settings for the application folder to stop this behavior.

## 📈 Performance Tips

For the best experience, optimize your hardware settings. Turn off overlays from other applications like Discord or Steam when you play. Disabling these overlays reduces the load on your graphics card and allows the detection loop to run faster. A faster detection loop results in smoother tracking of targets. If you play at 144Hz or higher, ensure you set your display settings to match that refresh rate inside the tool options. This helps the mouse-move logic synchronize with your screen updates. If you play on a laptop, plug it into a power source. Battery mode limits the performance of your processor and graphics chip, which can hinder the target tracking speed.
